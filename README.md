# ecommerce-clothing

Minimal skeleton of an e-commerce clothing project (React frontend + Node.js backend).

## Structure
- client/: React app (minimal files)
- server/: Node.js/Express API (minimal files)

## How to run (development)
- Client:
  ```
  cd client
  npm install
  npm start
  ```
- Server:
  ```
  cd server
  npm install
  node server.js
  ```

This zip contains skeleton files to get you started. Implement full features (DB, auth, API endpoints, build tooling) as needed.
